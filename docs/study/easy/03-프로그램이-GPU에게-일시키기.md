# 3장. 프로그램이 GPU에게 일을 시켜요 🍽️

> 이 장에서 배울 것: 프로그램은 GPU에게 "자리 좀 줘!"라고 주문해요. 식당 주문서처럼요.

## 🎯 오늘의 목표

1. "프로그램"이 뭔지 알기
2. 프로그램이 GPU에게 어떻게 부탁하는지 알기
3. "메모리 좀 줘"라는 주문이 뭔지 알기

---

## 프로그램이 뭐예요?

**프로그램**은 컴퓨터한테 시키는 **할 일 목록**이에요.
"이거 해, 저거 해" 하고 적어둔 **약속 목록** 같은 거예요.

예를 들어 그림을 그리는 프로그램, 게임 프로그램, AI 프로그램이 있어요.
AI 프로그램은 GPU의 도움을 많이 받아요. (1장 기억나죠? 일꾼이 많아서요.)
예: 글을 써 주는 챗봇 AI도 GPU 없이는 너무 느려요.

```
   프로그램 (할 일 목록)
  ┌──────────────────┐
  │ 1. GPU 자리 받기  │
  │ 2. 계산하기       │
  │ 3. 자리 돌려주기  │
  └──────────────────┘
```

---

## GPU에게 "메모리 좀 줘!" 주문하기

프로그램이 GPU에서 일을 하려면 **책상 자리(메모리)**가 필요해요.
(2장에서 메모리는 "물건 올려두는 큰 책상"이라고 했죠?)

그래서 프로그램은 GPU에게 이렇게 말해요.
**"책상 자리 2칸만 줘!"**

이 부탁을 컴퓨터 말로 **`cudaMalloc`**(쿠다말록)이라고 해요.
어려워 보이죠? 그냥 **"GPU야, 메모리 자리 좀 줘"라는 주문**이라고 알면 돼요.

```
  프로그램: "cudaMalloc — GPU야, 2칸 줘!"
       │
       ▼
  GPU: "그래, 여기 2칸 줄게~" 📦📦
```

---

## 식당 주문서처럼 생각해 봐요

식당에 가면 주문서를 써서 주문하죠?
"비빔밥 하나 주세요!"

GPU도 똑같아요. 프로그램이 **주문서**를 써서 GPU에게 줘요.
- 주문 내용: **"메모리 자리 주세요"** (이게 `cudaMalloc`)
- 다 쓰면: **"자리 돌려드릴게요"** (이건 `cudaFree` — 쿠다프리)

```
   주문서 (cudaMalloc)
  ┌────────────────────┐
  │ 주문: 메모리 2칸    │
  │ 받는 곳: GPU       │
  └────────────────────┘
        │
        ▼ (주방으로)
      GPU 가 자리 마련

   다 쓴 뒤 (cudaFree)
  ┌────────────────────┐
  │ "자리 돌려드려요!"  │
  └────────────────────┘
```

이렇게 **주문하기**(`cudaMalloc`)와 **돌려주기**(`cudaFree`)를 반복하면서
프로그램이 GPU를 써요.

---

## 📦 한 줄 정리

프로그램은 GPU에게 **주문서**를 줘요.
"메모리 자리 줘"가 `cudaMalloc`, "돌려줄게"가 `cudaFree`예요.

---

## ❓ 스스로 물어보기

**Q1. 프로그램이 GPU에게 "자리 줘"라고 하는 주문 이름은?**
👉 `cudaMalloc`이에요.

**Q2. 다 쓰고 자리를 돌려주는 주문 이름은?**
👉 `cudaFree`예요.
