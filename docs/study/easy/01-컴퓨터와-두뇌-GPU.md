# 1장. 컴퓨터와 두뇌, 그리고 GPU 🧠

> 이 장에서 배울 것: 컴퓨터 안에는 "두뇌"가 있어요. 그리고 GPU는 아주 특별한 두뇌예요.

## 🎯 오늘의 목표

1. 컴퓨터에도 "생각하는 부분"이 있다는 걸 알기
2. CPU랑 GPU가 뭐가 다른지 알기
3. 왜 GPU가 어떤 일은 훨씬 빠른지 알기

---

## 컴퓨터 안에는 작은 두뇌가 있어요

사람은 머리로 생각하죠?
컴퓨터도 똑같아요. 컴퓨터 안에는 **생각하고 계산하는 부분**이 있어요.

그 부분을 **CPU**라고 불러요.
(CPU = 컴퓨터의 똑똑한 두뇌, 라고 기억하면 돼요.)

```
   컴퓨터
  ┌──────────────┐
  │   🧠 CPU     │  ← 생각하고 계산하는 두뇌
  │              │
  └──────────────┘
```

CPU는 아주 똑똑해요. 어려운 문제도 척척 풀어요.
하지만 CPU는 일꾼이 **몇 명밖에 없어서** 한 번에 한 가지씩 순서대로 풀어요.
아주 똑똑한 선생님 몇 분이 문제를 푸는 것 같아요.

---

## GPU는 그림을 잘 그리는 특별한 두뇌예요

컴퓨터에는 두뇌가 하나 더 있을 수 있어요.
바로 **GPU**예요.

GPU는 원래 **그림과 영상을 빠르게 그리려고** 만든 두뇌예요.
게임 화면이나 영상이 부드럽게 움직이는 건 GPU 덕분이에요.

```
   컴퓨터
  ┌──────────────────────────┐
  │   🧠 CPU      🎨 GPU      │
  │  똑똑한 한명   그림 잘그리는│
  │              일꾼 아주많음 │
  └──────────────────────────┘
```

그런데 재미있는 점!
GPU 안에는 **작은 일꾼이 아주아주 많이** 들어 있어요.
한 명 한 명은 그렇게 똑똑하지 않아요.
하지만 **숫자가 엄청 많아서**, 비슷한 일을 한꺼번에 하면 정말 빨라요.

---

## 한 명이 1000문제 vs 1000명이 1문제씩

비유로 생각해 봐요.

산수 문제가 **1000개** 있어요. 다 비슷한 더하기 문제예요.

**CPU 방식** — 똑똑한 선생님 몇 명이 한 줄로 서서 순서대로 풀어요.
1번, 2번, 3번… 한 문제씩 차례로. 1000번 반복해야 해요.

```
CPU (선생님 몇 명):
  [➀] [➁] [➂] [➃] [➄] ...  (한 번에 몇 문제씩, 순서대로)
```

**GPU 방식** — 친구 1000명이 동시에 한 명씩 한 문제만 풀어요.
다 같이 동시에 풀어요. 그러면 한 번에 끝나요!

```
GPU (친구 1000명, 동시에!):
  [➀][➁][➂][➃][➄][➅][➆][➇][➈][➉]...  (빽빽이 동시에)
  [⑪][⑫][⑬][⑭][⑮][⑯][⑰][⑱][⑲][⑳]...
  ...  (한 번에 끝!)
```

비슷한 문제가 아주 많을 때는 **GPU가 훨씬 빨라요.**
그래서 요즘 인공지능(AI)을 가르칠 때 GPU를 많이 써요.
AI 공부는 비슷한 계산을 **엄청나게 많이** 하거든요.
예를 들어 사진 속 고양이를 알아맞히거나, 챗봇이 글을 쓰는 것 같은 일이에요.

> (사실은 GPU도 아무 일이나 다 빠른 건 아니에요. *비슷한 일이 아주 많을 때만* 빨라요. 지금은 이렇게 알면 충분해요.)

---

## 📦 한 줄 정리

CPU는 똑똑한 한 명, GPU는 평범하지만 **아주 많은 일꾼**.
비슷한 일이 많을 때는 일꾼이 많은 GPU가 빨라요.

---

## ❓ 스스로 물어보기

**Q1. CPU랑 GPU 중에 "일꾼이 많은" 쪽은 누구일까요?**
👉 GPU예요. 작은 일꾼이 아주 많아요.

**Q2. 비슷한 문제 1000개를 풀 때 GPU가 빠른 이유는?**
👉 일꾼이 많아서 **한꺼번에** 풀 수 있으니까요.

**Q3. 똑똑한 한 명이 어려운 문제 하나를 푸는 건 누가 더 잘할까요?**
👉 CPU예요. 한 가지 어려운 일은 똑똑한 한 명이 나아요.
