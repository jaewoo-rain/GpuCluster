# fGPU 교과서

**NVIDIA GPU 한 장을 여러 컨테이너가 나눠 쓰도록, 각자에게 메모리 사용 한도(quota)를 정해주고 강제하는 시스템** — 그 원리를 밑바닥부터 배우는 학습 문서입니다.

LD_PRELOAD로 `cudaMalloc`을 가로채 GPU 메모리를 분수(0.4 / 0.6처럼)로 쪼개 쓰는 프로토타입(Backend.AI의 fractional GPU를 재현한 캡스톤/연구용)입니다.

## 🧭 이 프로젝트를 한눈에

이 프로젝트는 크게 세 덩어리이고, 자원을 지키는 "세 문지기"가 협력합니다.

| 문지기 | 언제 검사하나 | 무엇을 막나 | 어디에 있나 |
|--------|----------------|-------------|-------------|
| **① 어드미션** | 컨테이너를 만들기 직전 | 지분 합이 1.0 초과 | 백엔드 (Python) |
| **② 훅** | 메모리를 요청할 때마다 | 컨테이너 한도 초과 | `libfgpu.so` (C) |
| **③ 드라이버** | 물리적 한계 | 진짜 GPU 메모리 바닥 | NVIDIA 드라이버 |

!!! tip "빠른 시작"
    시간이 없다면 온보딩의 **0장 → 4장 → 5장 → 8장 → 10장**만 읽어도 프로젝트의 뼈대가 잡힙니다.
