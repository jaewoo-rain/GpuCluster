# 5장. 메모리 칸 나눠 주기 (quota) 🗄️

> 이 장에서 배울 것: 각자 정해진 양만 쓰게 해요. 사물함 칸처럼, 내 칸을 넘으면 안 돼요.

## 🎯 오늘의 목표

1. "quota(쿼타)"가 무슨 뜻인지 알기
2. 비율(0.4, 0.6)이 실제로 몇 칸인지 알기
3. 칸을 넘으면 어떻게 되는지 알기

---

## 학교 사물함 이야기

학교에는 사물함이 있죠?
사물함에는 **칸**이 있어요.
내 칸은 내 것이고, 친구 칸은 친구 것이에요.

내가 아무리 물건이 많아도,
**내 칸을 넘쳐서 친구 칸까지 쓰면 안 돼요.**

GPU 메모리도 똑같아요.
각자 정해진 칸만큼만 쓰게 해요.
이 "정해진 칸"을 **quota(쿼타)**라고 불러요.
쉬운 말로 하면 **"내 몫"** 이에요.

```
   GPU 메모리 사물함
  ┌──────────────┬──────────────┐
  │  친구 A 칸   │  친구 B 칸   │
  │  (quota 0.4) │  (quota 0.6) │
  │              │              │
  └──────────────┴──────────────┘
     A는 이 안에서만   B는 이 안에서만
```

---

## 비율이 실제로 얼마예요?

2장에서 GPU 메모리를 "12칸짜리 큰 책상"이라고 했죠?
(실제로는 8기가바이트(GB)짜리 GPU가 흔해요. 그냥 "아주 큰 책상"으로 알면 충분해요.)

**0.4(비율)** 이면:
전체 책상 크기 × 0.4 = **내 몫**

예를 들어 전체가 8GB이면:
```
8GB × 0.4 = 3.2GB  ← 친구 A의 몫
8GB × 0.6 = 4.8GB  ← 친구 B의 몫
```

합치면 `3.2 + 4.8 = 8GB` — GPU를 빈틈 없이 꽉 채워서 나눠요!

```
  전체 GPU 메모리 (8GB = 100%)

  ┌──────────────────────────────────┐
  │  친구 A (0.4)  │  친구 B (0.6)  │
  │   3.2GB까지    │   4.8GB까지    │
  └──────────────────────────────────┘
         3.2 + 4.8 = 8  (딱 맞게 꽉!)
```

## 급식 정량처럼 생각해요

학교 급식도 **정량**이 있죠?
밥을 한 숟가락만 더 달라고 해도 아주머니가 안 주실 때가 있잖아요.
"오늘은 이만큼씩만이야!"

그것처럼 GPU도:
- 각자 정해진 양(quota)이 있어요.
- 그 양을 넘어서 달라고 하면 → **DENY(거절)**!

```
급식 아주머니 (= 가로채기 도구 libfgpu.so):

  "밥 3그릇 줘요!"  →  "안 돼! 너는 1그릇까지야 (DENY)"
  "밥 반 그릇 줘요" →  "그래, 여기 있어! (ALLOW)"
```

---

## 쓰다가 돌려주면 다시 쓸 수 있어요

사물함도 마찬가지죠?
내 물건을 꺼내면 칸에 자리가 생기잖아요.
그럼 나중에 다른 물건을 넣을 수 있어요.

GPU도 똑같아요.
- 주문: "2칸 줘" → 2칸이 줄어요.
- 돌려줌: "2칸 반납" → 2칸이 다시 생겨요.

```
처음:    [내 몫: 5칸 남아]
주문 2칸: [내 몫: 3칸 남아]
주문 2칸: [내 몫: 1칸 남아]
주문 2칸: → "칸이 부족해! DENY" ← 1칸밖에 없는데 2칸 달라고 했으니
반납 2칸: [내 몫: 3칸 남아]
```

## 합이 1.0이면 딱 맞게 나눈 거예요

0.4 + 0.6 = 1.0
이건 GPU 전체를 **빈틈 없이 딱 맞게** 나눈 거예요.

합이 1.0보다 크면 → 사실은 너무 많이 나눠준 것!
(예: 0.7 + 0.7 = 1.4 → 전체보다 40%나 더 나눠주려 함)
이럴 때는 누군가가 미리 막아줄 수 있어요.
그 역할을 하는 것을 **입장 조절**(나중에 7장에서 자세히 배워요!)이라고 불러요.
지금은 그냥 "누군가 미리 막아줘요"로만 알면 돼요.

```
  좋은 예:  0.4 + 0.6 = 1.0  (딱 맞음)
  나쁜 예:  0.7 + 0.7 = 1.4  (합이 1 초과 — 접수처에서 막음)
```

---

## 📦 한 줄 정리

quota는 "내 몫"이에요.
비율(0.4, 0.6)을 전체 GPU 크기에 곱하면 내 몫이 나오고,
그 몫을 넘으면 주문이 거절돼요.

---

## ❓ 스스로 물어보기

**Q1. quota를 쉬운 말로 바꾸면?**
👉 "내 몫"이에요. 내가 쓸 수 있는 최대 양이에요.

**Q2. GPU가 8GB인데 ratio(비율)가 0.4면 내 몫은 몇 GB예요?**
👉 8 × 0.4 = 3.2GB예요.

**Q3. 내 몫이 3.2GB인데, 4GB를 달라고 하면 어떻게 돼요?**
👉 DENY! 4 > 3.2이니까 거절돼요.
