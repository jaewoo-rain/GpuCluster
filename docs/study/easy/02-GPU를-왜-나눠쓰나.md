# 2장. GPU를 왜 나눠 쓸까요? 🍰

> 이 장에서 배울 것: GPU는 크고 비싼데, 한 명이 다 못 써요. 그래서 나눠 쓰면 좋아요.

## 🎯 오늘의 목표

1. GPU가 왜 "아깝게" 쓰일 수 있는지 알기
2. 나눠 쓰면 왜 좋은지 알기
3. "분수처럼 쪼개기"가 무슨 말인지 알기

---

## GPU는 크고 비싸요

지난 장에서 GPU는 일꾼이 아주 많은 두뇌라고 했죠?
그런데 이 GPU는 **아주 비싸요.** 그리고 **아주 커요.**

GPU 안에는 **메모리**라는 게 있어요.
메모리는 일을 하려고 **물건을 잠깐 올려두는 큰 책상**이라고 생각해요.

```
   GPU 의 책상 (메모리)
  ┌────────────────────────────┐
  │                            │
  │      아주 넓은 책상!        │
  │   (예: 12칸짜리 큰 책상)    │
  │                            │
  └────────────────────────────┘
```

이 책상은 아주 넓어요. 예를 들면 **12칸**짜리예요.
(진짜로는 "12 기가바이트" 정도예요. 그냥 "아주 큰 책상"이라고 알면 돼요.)

---

## 한 명이 다 못 써요

그런데 문제가 있어요.
어떤 친구는 GPU에서 **작은 일**만 해요.
12칸짜리 큰 책상에서 **2칸**만 쓰고 나머지는 비어 있어요.

```
   친구 한 명이 쓸 때
  ┌────────────────────────────┐
  │ 📦📦                       │  ← 2칸만 씀
  │      (나머지 10칸은 텅 빔)  │
  │                            │
  └────────────────────────────┘
         아깝다! 😢
```

10칸이 그냥 놀고 있어요. 너무 아깝죠?
비싼 GPU인데 거의 안 쓰고 있는 거예요.

---

## 나눠 쓰면 좋아요!

좋은 생각이 있어요.
큰 책상을 **여러 친구가 나눠 쓰면** 되잖아요!

```
   여러 명이 나눠 쓸 때
  ┌────────────────────────────┐
  │ 📦📦📦📦📦 │ 📗📗📗📗📗📗📗  │
  │  친구 A   │   친구 B       │
  └────────────────────────────┘
       꽉 차게 잘 씀! 😄
```

큰 운동장을 떠올려 봐요.
한 사람만 쓰면 운동장이 텅텅 비어요.
여러 반이 구역을 나눠 쓰면 운동장을 알차게 쓰죠!

큰 케이크도 마찬가지예요.
혼자 다 못 먹으면 친구들이랑 나눠 먹으면 돼요. 🍰

---

## "분수처럼 쪼개기"가 뭐예요?

여기서 중요한 말이 나와요. **fractional**(프랙셔널)이에요.
어려운 말 같지만 뜻은 쉬워요. **"비율로 쪼갠다"**는 뜻이에요.

`0.4`, `0.6` 같은 **점이 찍힌 숫자**를 본 적 있나요? (어려운 말로 '소수'라고 해요. 2, 3, 5, 7 같은 수학 시간의 소수(素數)와는 다른 말이에요. 여기서는 그냥 "점이 찍힌 숫자"라고 기억해요.)
- `0.4`는 100칸 중 40칸 = **전체의 40%** — 큰 책상의 거의 절반보다 조금 적게
- `0.6`은 100칸 중 60칸 = **전체의 60%** — 큰 책상의 절반보다 조금 많게

```
   12칸 책상을 0.4 / 0.6 으로 나누기
  ┌────────────────────────────┐
  │  친구 A (0.4)  │ 친구 B (0.6)│
  │  ←약 5칸→      │ ←약 7칸→    │
  └────────────────────────────┘
   합치면 0.4 + 0.6 = 1.0 (전체 다 씀!)
```

이렇게 GPU를 분수처럼 쪼개서 나눠 주는 게 이 프로그램이 하는 일이에요.
우리 프로그램 이름이 **fGPU**인데, 앞의 **f**가 바로 이 "분수(fractional)"의 f예요!

> (사실은 책상이 정확히 칸으로 나뉘진 않지만, "비율로 나눈다"는 느낌만 알면 충분해요.)

---

## 📦 한 줄 정리

GPU는 크고 비싼데 한 명은 조금만 써요.
그러니 **분수처럼 쪼개서** 여러 명이 나눠 쓰면 알차게 쓸 수 있어요.

---

## ❓ 스스로 물어보기

**Q1. 친구 한 명이 큰 GPU를 조금만 쓰면 뭐가 문제예요?**
👉 나머지가 놀아서 아까워요.

**Q2. `0.4`랑 `0.6`을 더하면 얼마예요? 그건 무슨 뜻이에요?**
👉 `1.0`이에요. GPU 전체를 다 쓴다는 뜻이에요.

**Q3. "fractional(비율로 쪼개기)"은 무슨 뜻이에요?**
👉 GPU를 비율(0.4, 0.6 같은 점이 찍힌 숫자)로 나눠 쓴다는 뜻이에요. 0.4는 40%예요.
