# 🎈 아주 쉬운 GpuCluster 교과서

안녕하세요! 이 책은 **GPU를 여러 명이 나눠 쓰는 신기한 프로그램**에 대한 이야기예요.

코딩을 한 번도 안 해봤어도 괜찮아요. 컴퓨터를 잘 몰라도 괜찮아요.
저랑 같이 **그림과 비유**로 천천히 알아가 봐요. 😊

## 📖 이 책은 누구를 위한 걸까요?

- 컴퓨터가 어떻게 일하는지 궁금한 친구
- "GPU"라는 말을 들어봤지만 뭔지 모르는 친구
- 어려운 책(어른용)을 보고 "으악 너무 어려워!" 했던 친구

이 책은 **어려운 말을 안 써요.**
꼭 필요한 어려운 말이 나오면, 먼저 **우리가 아는 것**(급식, 사물함, 운동장 같은 것)에 빗대서 설명할 거예요.
