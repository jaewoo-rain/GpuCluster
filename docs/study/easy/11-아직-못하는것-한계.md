## 솔직하게 이야기해요

어려워 보이죠? 사실 별거 아니에요.
우리 프로그램이 못 하는 것들이 있어요.
그걸 숨기는 게 아니라 **솔직하게** 말하는 게 좋아요.

과학에서도 "이건 됐고, 이건 아직 안 됐어"라고 말하는 게 더 믿음이 가잖아요.
우리도 그렇게 할 거예요.

---

## 못 하는 것 1: 계산 일꾼을 진짜로 칸막이 치지 못해요

10장에서 배운 것 기억하죠?

GPU 안에는 **계산 일꾼들**이 있어요.
우리는 **메모리(책상 자리)** 는 잘 나눠줄 수 있어요.

하지만 **계산 일꾼들** 에 칸막이를 치진 못해요.

```
  우리가 할 수 있는 것:
  ┌──────────────────────────────────┐
  │  메모리:  A 칸 │ B 칸  ← 잘 됨! │
  └──────────────────────────────────┘

  우리가 못 하는 것:
  ┌──────────────────────────────────┐
  │  계산 일꾼: 모두 함께 쓰는 상태  │
  │  ← 칸막이 없음                   │
  └──────────────────────────────────┘
```

그래서 A와 B가 동시에 열심히 일하면
계산 일꾼들을 두고 경쟁하게 돼요.
둘 다 조금 느려질 수 있어요.

**왜 못 하나요?**
계산 일꾼을 칸막이로 나누는 건 **특별한 하드웨어 기능**(MIG라고 해요)이 있어야 해요.
그런데 우리가 쓰는 GPU(RTX 4060, 4070 같은 일반 게임용 GPU)는 그 기능이 없어요.
그래서 소프트웨어로는 아무리 노력해도 못 해요.

> (번갈아 쓰기 throttle로 어느 정도 도움이 되지만, 진짜 칸막이는 아니에요.)

---

## 못 하는 것 2: 규칙을 안 지키기로 마음먹은 프로그램은 못 막아요

우리 가로채기 도구(`libfgpu.so`)는 **LD_PRELOAD**를 이용해요.
(4장에서 배운 "가짜 점원을 먼저 세우는 방법"이요.)

그런데 만약 어떤 프로그램이 **가짜 점원을 무시하고** 직접 주방에 들어가려 하면?
우리는 막을 수 없어요.

예를 들어:
- 일반적인 AI 프로그램 → 우리 도구가 잘 가로채요.
- 규칙을 무시하고 뒷문으로 몰래 들어가는 프로그램 → 우리 도구를 피해 갈 수 있어요.

```
  일반 프로그램:
  프로그램 → [가짜 점원 libfgpu.so] → GPU
             (잘 가로챔!)

  규칙 안 지키는 프로그램:
  프로그램 ─────────────────────────→ GPU
             (가짜 점원 무시!)
```

그래서 이 프로그램은 **"서로 믿고 협력한다"는 가정** 아래서 동작해요.
친한 친구들끼리, 또는 같은 학교 친구들끼리 나눠 쓰는 상황이에요.
일부러 규칙을 깨려는 나쁜 프로그램은 막지 못해요.

---

## 못 하는 것 3: 완벽한 하드웨어 격리는 불가능해요

아주 비싼 전문가용 GPU(A100, H100 같은 것)는
**MIG**(엠아이지)라는 기능이 있어요.
MIG가 있으면 GPU를 **하드웨어 수준에서** 완전히 나눌 수 있어요.
계산 일꾼도, 메모리도, 완벽하게 따로따로.

우리가 쓰는 GPU에는 그 기능이 없어요.
그래서 우리는 소프트웨어로 최대한 흉내를 낼 뿐이에요.

```
  완벽한 하드웨어 격리 (MIG):
  ┌──────────────────────────────┐
  │ A 전용 GPU 조각 │ B 전용 조각 │
  │ (메모리+일꾼 완전 분리)       │
  └──────────────────────────────┘
  → 아주 비싼 GPU에서만 가능

  우리 방식 (소프트웨어):
  ┌──────────────────────────────┐
  │ 메모리: A 칸 / B 칸 (잘 됨)  │
  │ 일꾼: 함께 씀 (분리 불가)     │
  └──────────────────────────────┘
  → 일반 게임용 GPU에서도 됨!
```

---

## 그래도 충분히 쓸모 있어요!

못 하는 것도 있지만, 잘 하는 것도 분명히 있어요.

```
  우리가 잘 하는 것:
  ✓ 메모리(책상 자리) 나누기
  ✓ 각자 몫을 넘으면 정확히 거절하기
  ✓ 여러 명이 동시에 방을 쓸 수 있게 하기
  ✓ 방 신청, 확인, 삭제를 쉽게 하기
  ✓ 번갈아 쓰기로 어느 정도 공평하게 하기
```

일반 게임용 GPU(RTX 4060, 4070)로도 여러 명이 나눠 쓸 수 있어요.
이건 아무것도 없을 때보다 훨씬 낫죠!

학교 친구들끼리 GPU를 나눠 쓰고 싶을 때,
또는 회사 동료들끼리 같은 GPU를 쓰고 싶을 때,
이 프로그램이 도움이 돼요.

---

## 📦 한 줄 정리

메모리 나누기는 잘 돼요.
하지만 계산 일꾼 칸막이, 나쁜 프로그램 막기, 완벽한 하드웨어 격리는 못 해요.
그래도 일반 GPU를 나눠 쓰는 데는 충분히 쓸모 있어요!

## ❓ 스스로 물어보기

**Q1. 우리 프로그램이 잘 나눠주는 건 뭐예요?**
👉 메모리(책상 자리)예요. 각자 몫을 잘 지켜줘요.

**Q2. 계산 일꾼을 진짜로 나눠주려면 뭐가 필요해요?**
👉 MIG라는 특별한 하드웨어 기능이 필요해요. 일반 GPU에는 없어요.

**Q3. 이 프로그램은 어떤 상황에 쓸모 있어요?**
👉 친구들이나 동료들끼리 일반 GPU를 나눠 쓸 때요. 서로 믿는 상황에서요.
